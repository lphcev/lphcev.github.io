HTML 学习笔记

發表於2001-01-01|更新於2024-03-05|Front
|閱讀量:

HTML 全称 Hyper Text Markup Language (超文本标记语言), 是一种用来结构化 Web 网页及其内容的标记语言. 它包括一系列标签, 通过这些标签可以将网络上的文档格式统一, 使分散的 Internet 资源连接为一个逻辑整体
