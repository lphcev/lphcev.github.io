Git 教程

Git 是一个开源的分布式版本控制系统，可以有效、高速地处理从很小到非常大的项目版本管理。也是 Linus Torvalds 为了帮助管理 Linux 内核开发而开发的一个开放源码的版本控制软件

Github

添加 SSH 公钥

登录 Github 账号, 访问 https://github.com/settings/ssh/new

将 C:\Users\XYZ\.ssh\id_rsa.pub 内容拷贝到文本框中, 点击 Add SSH key 按钮, 完成配置

测试 SSH 连接

Github Pages
