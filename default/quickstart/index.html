Markdown

Markdown 是一种轻量级标记语言，它使用简单的纯文本格式语法，可快速转换为结构化的 HTML 或其他格式文档。其核心设计理念是易读易写，适合编写文档、笔记、博客甚至书籍

为什么选择 Markdown？

  1. 轻量高效：无需复杂排版工具，纯文本即可编写
  2. 跨平台兼容：可在任何文本编辑器中编写，支持 GitHub、博客平台等
  3. 专注内容：通过简单符号实现排版，减少对格式的干扰
  4. 灵活扩展：支持 HTML 嵌入，并衍生出多种扩展语法（如表格、流程图）
