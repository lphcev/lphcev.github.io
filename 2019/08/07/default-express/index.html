基本使用

路由

路由是指确定应用程序如何响应对特定端点的客户端请求，该端点是 URI（或路径）和特定的 HTTP 请求方法 GET/POST 等

每个路由可以有一个或多个处理程序函数，这些函数在路由匹配时执行

获取请求参数

通过 query 属性获取 URL 的查询参数
