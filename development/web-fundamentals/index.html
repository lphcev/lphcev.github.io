闭包 JavaScript 中闭包（Closure）是指一个函数能够访问并记住其词法作用域，即使该函数在其词法作用域之外执行。简单来说，闭包允许函数访问其定义时所处的环境中的变量，即使这个函数在定义环境之外被调用 闭包形成条件 函数嵌套 一个函数（内部函数）定义在另一个函数（外部函数）的内部 内部函数可以访问外部函数的变量和参数 内部函数被外部函数以外的作用域引用 内部函数必须被外部函数以外的作用域引用（例如，被返回、赋值给全局变量、作为回调函数传递等） 只有这样，内部函数才会保留对外部函数作用域的引用，从而形成闭包 闭包的关键点 词法作用域：闭包是基于词法作用域的，即函数在定义时就已经确定了它可以访问的作用域 作用域链：内部函数会保留对外部函数作用域的引用，即使外部函数已经执行完毕 内存管理：闭包会阻止外部函数的作用域被垃圾回收，因为内部函数仍然持有对它的引用 案例：计数器 创建一个变量 count，初始值为 0，创建一个函数 increment，用于对这个变量进行增量 12345678let count = 0;function...
