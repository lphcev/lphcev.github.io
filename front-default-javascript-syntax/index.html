JavaScript 基础语法

發表於2003-01-02|更新於2024-03-26|Front
|閱讀量:

本章讲解 JavaScriot 的一些基础语法和特性, 也是学习的第一章节

理解三层关系

  • HTML 是一种标记语言, 用来结构化我们的网页内容并赋予内容含义

  • CSS 是一种样式规则语言, 可将样式应用于 HTML 元素

  • JavaScript 是一种脚本语言, 可以用于动态交互

一个网页通过 HTML 元素来定义基本的结构, 由 CSS 来定义这块结构的样式风格, 从而增强表现形式, 最后通过 JavaScript 来定义这块内容的动态内容 (例如响应用户操作, 数据校验等)

注释

注释就是对代码的的解释和说明, 其目的是让人们能够更加轻松地了解代码, 提高程序代码的可读性. 同时, 注释不会被计算机编译

JavaScript 中支持如下三种注释, 分别是单行注释, 多行注释和文档注释

常用数据类型

  • 数字 (number) : 整数或浮点数
  • 字符串 (string) : 一串表示文本值的字符序列
  • 布尔 (boolean) : 一个条件判断的结果
  • 对象 (object) : 用于存放各种具有键/值对的实体

除上述几种数据类型以外, 还有一些比较特殊的值

  • 未定义值 (undefined) : 当定义一个变量且未赋值时的默认情况
  • 空值 (null) : 表示一个对象的指针为空
  • 非数字 (NaN) : 数字类型的特殊值, 用于表示某个值不是数字

概念术语

标识符

标识符是用来标识变量, 函数或属性的字符序列, 上面定义的变量名 say 就是一个标识符

JavaScript 中的标识符有着自己的命名规范

  • 标识符必须以字母, 下划线 _ 或美元符号 $ 开头
  • 标识符不能以数字开头
  • 标识符通常使用小驼峰命名法

字面量

字面量是脚本中按字面意思给出的固定的值, 上面定义的变量值 'Hello' 就是一个字面量

表达式

表达式是由数字, 运算符, 括号, 自由变量和约束变量等以能求得数值的有意义排列方法所得的组合. 约束变量在表达式中已被指定数值, 而自由变量则可以在表达式之外另行指定数值

逻辑运算符

逻辑运算符通常用于在多个操作数之间进行逻辑运算, 返回一个布尔值

语句块

JavaScript 提供一套灵活的语句集, 特别是控制流语句, 你可以用它在你的应用程序中实现大量的交互性功能

在 JavaScript 中, 任何表达式都可以看作一条语句

最基本的语句是用于组合语句的语句块. 该块由一对大括号界定

switch 语句

switch 语句允许一个程序求一个表达式的值并且尝试去匹配表达式的值到一个 case 标签. 如果匹配成功, 这个程序执行相关的语句

程序首先查找一个与 num 匹配的 case 语句, 然后将控制权转移到该子句, 执行相关的语句. 如果没有匹配值, 程序会去找 default 语句, 如果找到了, 控制权转移到该子句, 执行相关的语句. 如果没有找到 default, 程序会继续执行 switch 语句后面的语句. default 语句通常出现在 switch 语句里的最后面, 当然这不是必须的

break 语句与每个 case 语句相关联, 保证在匹配的语句被执行后程序可以跳出 switch 并且继续执行 switch 后面的语句. 如果 break 被忽略, 则程序将继续执行 switch 语句中的下一条语句

循环语句

循环提供了一种快速和简单的方式去做一些重复的事. 循环有很多种类, 但本质上它们都做的是同一件事 : 它们把一个动作重复了很多次 (实际上重复的次数有可能为 0). 各种循环机制提供了不同的方法去确定循环的开始和结束. 不同情况下, 某一种类型循环会比其他的循环用起来更简单

for 语句

一个 for 循环会一直重复执行, 直到指定的循环条件为 false

当一个 for 循环执行的时候, 会发生以下过程

  1. 如果有初始化变量或表达式 var num = 1, 它将被执行. 这个表达式通常会初始化一个或多个循环计数器, 但语法上是允许一个任意复杂度的表达式的
  2. 计算 num <= 5 表达式的值. 如果为 true, 循环中的语句会被执行. 如果为 false, for 循环终止. 如果这个表达式整个都被省略掉了, 则会被默认为 true
  3. 循环中的 console.log(num) 被执行
  4. 如果有更新表达式 num++, 执行更新表达式
  5. 回到步骤 2
